# Image Watermark Remover — MVP 需求文档

---

## 1. 产品概述

| 项目 | 内容 |
|------|------|
| **产品名称** | WatermarkGone（待定） |
| **一句话描述** | 用户通过涂抹水印区域，一键智能移除图片水印 |
| **核心价值** | 简单、快速、免费，无需下载软件 |
| **目标用户** | 内容创作者、设计师、普通用户，需要快速去除图片水印 |
| **目标市场** | 境外用户为主（欧美、东南亚） |

---

## 2. 功能列表

### MVP 必做功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 图片上传 | 拖拽或点击上传，支持 JPG/PNG/WebP | P0 |
| 水印涂抹 | 画笔涂抹水印区域，生成 mask | P0 |
| AI 去水印 | 调用 Replicate API 执行 inpainting | P0 |
| 结果预览 | 显示处理前后对比 | P0 |
| 图片下载 | 一键下载无水印结果图 | P0 |
| 响应式 UI | 适配桌面端和移动端 | P0 |

### 未来版本功能

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 批量处理 | 一次处理多张图片 | P1 |
| 撤销/重做 | 撤销涂抹操作 | P1 |
| 边缘羽化 | mask 边缘软化，减少生硬感 | P1 |
| 历史记录 | 本地保存最近处理记录 | P2 |
| 付费高清下载 | 原图尺寸下载（免费版限制分辨率） | P2 |

## 4. 技术选型

### 前端

| 组件 | 技术 | 说明 |
|------|------|------|
| 框架 | **Next.js** (App Router) | SSR + 静态导出，SEO 友好 |
| 样式 | **TailwindCSS** | 快速开发响应式 UI |
| 涂抹画布 | **Fabric.js** 或 **Konva.js** | 专业的 canvas 图像处理库 |
| 状态管理 | React `useState` / `useReducer` | MVP 阶段不需要 Redux |
| 部署 | **Cloudflare Pages** | 全球 CDN，免费额度充足 |

### 后端（Minimal）

| 组件 | 技术 | 说明 |
|------|------|------|
| API 路由 | **Next.js API Routes** 或 **Cloudflare Workers** | 无状态，按调用计费 |
| AI 模型 | **Replicate** (`kaiypert/remove-anything` 或 `stability-ai/sdxl-inpainting`) | 按次计费，无需 GPU |
| 前端代理 | 绕过 CORS，直接调用 Replicate（前端直接请求） | MVP 阶段可行 |

> **注意**：Replicate API 有 CORS 限制，如果浏览器直接调用失败，需要一个轻量后端做代理（Cloudflare Worker 即可，成本极低）。

## 6. 非功能性需求

| 项目 | 要求 |
|------|------|
| **加载速度** | 首屏 < 2s（Lighthouse Score > 90） |
| **图片限制** | 最大 10MB，支持 JPG/PNG/WebP |
| **处理时间** | 通常 5-15 秒（取决于 Replicate 模型） |
| **移动端适配** | 触屏涂抹支持，响应式布局 |
| **错误处理** | 文件过大、格式错误、API 失败等友好提示 |
| **隐私** | 图片不存储，处理完即释放（内存） |

## 8. 成本估算（MVP）

| 项目 | 方案 | 月成本 |
|------|------|--------|
| 前端托管 | Cloudflare Pages | **$0**（免费） |
| 域名 | Namecheap / Cloudflare Registrar | ~$10/年 |
| API 调用 | Replicate（按次） | ~$0.01-0.05/张 |
| CORS 代理 | Cloudflare Worker | **$0**（免费额度 100k 请求/月） |

> **预估**：前期几乎没有成本，纯免费栈跑 MVP。等用户量上来再考虑付费扩展。
